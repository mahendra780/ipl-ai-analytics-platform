🏏 IPL AI Analytics Platform

An AI-powered IPL analytics and win prediction platform built using Python, Pandas, Machine Learning, and Streamlit.

🚀 Live App

https://ipl-ai-analytics-platform-msz5hhxcgcwvysvs8vofrn.streamlit.app/

---

📌 Features

📊 Match Analytics

- Team performance analysis
- Toss impact analysis
- Venue analysis
- Season trends
- Match result analysis

🏏 Batter Analytics

- Top run scorers
- Strike rate analysis
- Most fours & sixes
- Batter profile system
- Season-wise career progression

🎯 Bowling Analytics

- Most wickets
- Best economy rates
- Dot ball analysis
- Powerplay bowling analytics
- Death over bowling analytics

🔥 Advanced Match Insights

- Highest team scores
- Highest successful chases
- Lowest defended totals
- Biggest win margins
- Closest matches

🤖 AI Win Prediction System

Machine Learning model predicts win probability based on:

- batting team
- target
- current score
- overs
- wickets left
- required run rate

Model Accuracy:

- ✅ 83.81%

- Dark premium theme

- Interactive charts
- Organized tab layout
- Modern Streamlit dashboard
